# Alt Start
A BOTW mod allowing you to start the game in alternate locations, such as regional villages.   
Starting Locations:   
 - Goron City
 - Gerudo Town
 - Rito Village
 - Zora's Domain
 - Hateno Village
 - Lurelin Village
 - Kakariko Village
 - Korok Forest

If Second Wind is installed, Ordon Village will also be available.
